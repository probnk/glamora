# ✨ Overview

**Glamora** is a next-generation fashion shopping application built using **Flutter** that transforms the traditional online shopping experience into an immersive and intelligent journey.

Instead of simply browsing products, customers can virtually try on clothing using **Google ARCore**, receive personalized recommendations, interact with an AI-powered shopping assistant, track orders in real time, and enjoy a seamless shopping experience powered by Firebase and Supabase.

The application is designed with scalability, performance, and modern UI/UX principles to provide a premium mobile commerce experience.

---

# 🎯 Why Glamora?

Traditional fashion shopping apps suffer from several challenges:

- High product return rates
- Customers cannot visualize clothing before purchasing
- Limited shopping assistance
- Poor personalization
- Weak customer engagement
- Generic recommendations

Glamora solves these problems by combining Artificial Intelligence with Augmented Reality to help customers make smarter purchasing decisions.

# 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/probnk/glamora.git
```

### Navigate

```bash
cd glamora
```

### Install Packages

```bash
flutter pub get
```

### Run

```bash
flutter run
```

---

# 🔑 Environment Variables

Create a `.env` file.

```env
FIREBASE_API_KEY=
SUPABASE_URL=
SUPABASE_ANON_KEY=
DEEPSEEK_API_KEY=
TRACKINGMORE_API_KEY=
```
